# Job Listing App

This is a job listing web application that allows users to browse available jobs, filter them based on various criteria, and navigate to different sections of the platform.

## Features

- **Find Jobs**: View all job listings with filtering options.
- **Employers**: Learn how employers can post jobs.
- **Admin**: Admin panel for managing job postings.
- **About Us**: Information about the platform.
- **Contact Us**: Reach out for inquiries.
- **Login**: Access user accounts.

## Running the App

### Prerequisites

Ensure you have the following installed:

- [Node.js](https://nodejs.org/) (Recommended LTS version)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. Clone the repository:

   ```sh
   git clone https://github.com/your-username/Job_Listing.git
   cd Job_Listing
   ```

2. Install dependencies:

```sh
 npm install
 # or
 yarn install
```

3. Start deployment server:

```
 npm run dev
 # or
 yarn dev
```

4. Open your browser and navigate to host assigned to your site e.g 'http://localhost:3000'

## Navigation and Page Links

- **Find Jobs** (/find-jobs)

- **About Us** (/about-us)
